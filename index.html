How I built it

  • Minimax AI

    I wrote a recursive search that simulates all future moves so the AI never loses in Hard mode.

  • Keyboard & Screen Readers

    I wanted anyone to be able to play, so I added full arrow-key and screen reader support.

  • Synthesized Audio

    Instead of downloading mp3s, I used code to synthesize sound waves directly in the browser.
