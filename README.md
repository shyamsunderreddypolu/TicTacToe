# Tic-Tac-Toe Game in Java

This project contains a console-based Tic-Tac-Toe game and a Swing-based GUI version.

## Features

- Two-player console mode
- Input validation for row and column values
- Occupied-cell protection
- Win and draw detection
- Replay support with score tracking
- GUI version with clickable cells

## How to Run

### Console version

```bash
javac src/TicTacToe.java
java -cp src TicTacToe
```

### GUI version

```bash
javac src/TicTacToeGUI.java
java -cp src TicTacToeGUI
```

## Technologies Used

- Java
- Scanner
- Swing

## Project Stages

1. Basic console Tic-Tac-Toe
2. Validated console Tic-Tac-Toe with score tracking
3. GUI-based Tic-Tac-Toe
